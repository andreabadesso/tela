# 055 — Per-User Budget & Rate Limiting

**Status**: pending
**Phase**: 6 — Enterprise
**Dependencies**: 050, 054
**Estimated effort**: medium (2-8h)

## Context

With 140 users, API costs can spiral. Need per-user and per-team budget controls, plus rate limiting on MCP tool usage.

## Deliverables

### Extend budget policies

The existing `budget_policies` table gets new scope options:

```sql
-- scope: 'agent', 'global', 'user', 'team', 'role'
-- scope_id: agent_id, null, user_id, team_id, role_id
```

### Budget enforcement flow

```
User sends message
  → Orchestrator.chat()
    → checkBudget(userId) — check user's personal budget
    → checkBudget(agentId) — check agent's budget
    → If either hits hard threshold → pause, create approval
    → If soft threshold → add warning to response
```

### Rate limiting dashboard

Admin UI page showing:
- Per-user: total API cost this month, requests today
- Per-team: aggregate cost
- Per-connection: MCP tool call counts
- Alerts for users approaching limits

### Cost allocation

After each agent `query()`:
- Log `cost_events` with: user_id, agent_id, input/output tokens, cost_cents
- Agent cost = sum of all users' calls to that agent
- User cost = sum of all their calls across all agents

### Admin budget management

```
GET    /api/admin/budgets           — list budget policies
POST   /api/admin/budgets           — create budget policy
PUT    /api/admin/budgets/:id       — update
DELETE /api/admin/budgets/:id       — delete
GET    /api/admin/budgets/usage     — usage summary (by user, team, agent)
```

### User-facing cost view

Each user can see their own usage in Settings:
- "Your usage this month: $X.XX / $Y.YY limit"
- Top agents used
- Tool call breakdown

## Acceptance Criteria

- [ ] Per-user budget policies enforceable
- [ ] Per-team budget policies enforceable
- [ ] Hard stop pauses user and creates approval
- [ ] Soft warning included in agent response
- [ ] Rate limiting on MCP tool calls per user per connection
- [ ] Cost breakdown visible per user, team, agent
